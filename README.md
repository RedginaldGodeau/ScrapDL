# ScrapDL
Un scrappeur qui télécharge toutes les images et vidéos d'un site.

## Installation
Si les fichiers dans bin/build ne s'exécutent pas, alors (nécessite Docker & Make) :
```shell
make build
```
## Exécution:
```shell
./bin/build/scrapDL(.exe) [Chemin de téléchargement] [URL]
```
